Objectives:
In this assignment, you will learn about programming with sockets by implementing a
modified version of the File Transfer Protocol (FTP) for transferring files between two
machines. 
The purpose of this assignment is to introduce you to the socket programming API as well as
a variety of operating system concepts that are widely used in the construction of networked
applications.  

Project program summry :

This program working as a file transfer server with one client or few clients at the same time.
server program to handle with using this command commands :

'USER' following username and password : Log In to the server
'LIST' for show list of file in the directory
'GET' for get file or open file
'PUT' for edit file or add a new file into the server
'DEL' for delete file
'QUIT' log out or exit from the prosess
